## 使用

#### cmd

    ./chord-cmd --help

    USAGE:
        chord-cmd [OPTIONS] --input <input> --output <output>

    FLAGS:
        -h, --help       Prints help information
        -V, --version    Prints version information

    OPTIONS:
        -i, --input <input>       input dir
        -o, --output <output>     output dir
        -l, --level <level>...    log level
